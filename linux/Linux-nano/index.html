编辑器退出/文件保存

见到左下方有一个^X 离开的字样, 不过不是给你点的, 而是热键提示
^X即键盘操作的Ctrl + X

如果在文件毫无改动的情况下Ctrl + X, 会直接跳出编辑器回到命令行.
如果文件有过改动,则下方会询问是否储存变更

根据提示, 键入Y以保存

一般是不会更改文件名的, 如果需要就直接更改就好了.
确认文件名,键入Enter,即回车键.
随后文件保存,界面回到命令行.
