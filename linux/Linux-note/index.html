全局安装 miniconda3

  1. bash安装
  2. 安装位置不要在用户名下，改为/opt/miniconda3/
  3. 修改安装目录权限为所有用户都可以写入sudo chmod ugo+w -R /opt/miniconda3/
  4. conda加入PATH变量：sudo vim /etc/profile/，加入PATH=/opt/miniconda3/bin:$PATH
  5. 重开终端窗口 / 重连服务器
  6. conda -v / echo $PATH 检查确认
